# ClosetSocial

Base inicial de la app iOS para la red social de ropa.

## Qué trae ya

- login y registro
- timeline de amigos y compras compartidas
- pestaña de armario
- pestaña de outfits
- pestaña de perfil
- cliente HTTP preparado para hablar con el backend Vapor

## Cómo abrirlo

1. Abre [ClosetSocial.xcodeproj](./ClosetSocial.xcodeproj) en Xcode.
2. Si quieres login real, arranca antes el backend de `ClosetSocialBackend`.

## Estado actual

- Swift 6
- strict concurrency
- sin `DispatchQueue`
- networking con `async/await`
